Flask IT介绍

Flask IT是基于Flask构建的一个IT管理网站。

  • 动态自动加载功能目录
  • 图形化功能管理
  • 账号管理（登录，添加账号，更新账号，找回密码）
  • 多权限组管理
安装方法

1. 安装依赖

pip install -r requirements.txt

2.导入数据库

mysql -uroot < db_schema/flaskit.sql

3.启动应用

python run.py

4.浏览器网问：http://127.0.0.1:5000
